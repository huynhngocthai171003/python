Python Programming Language

MODULE TRONG PYTHON

Trong Python, module và package là hai khái niệm cơ bản giúp tổ chức và quản lý mã nguồn một cách hiệu quả.

Module

Một module trong Python là một file chứa định nghĩa và câu lệnh Python. File này có phần mở rộng là .py. Module có thể chứa định nghĩa của hàm, lớp và biến, cũng như mã thực thi. Mục đích chính của việc sử dụng module là để tái sử dụng mã nguồn. Bạn có thể định nghĩa các hàm, lớp và biến trong một module và sau đó sử dụng chúng trong một chương trình khác thông qua việc import module đó.

Package

Package trong Python là một thư mục chứa một hoặc nhiều modules hay các package khác nhau, nó được tạo ra nhằm mục đích phân bố các modules có cùng chức năng hay một cái gì đó, để dễ quản lý source code.


Tạo 1 module

Để tạo 1 module, chúng ta viết mã bằng tập lệnh Python và lưu nó dưới dạng tệp .py. Tạo 1 một tệp có tên mymodule.py trong thư mục dự án.
1. Ví dụ: #mymodule.py def generate_full_name(firstname, lastname): space = ' ' fullname = firstname + space + lastname return fullname def sum_two_nums (num1, num2): return num1 + num2 gravity = 9.81 person = { "firstname": "Asabeneh", "age": 250, "country": "Finland", "city":'Helsinki' }


Nhập vào 1 module

Để nhập tệp, chúng ta sử dụng từ khóa import và tên của tệp
1. Ví dụ:
#main.py import mymodule print(mymodule.generate_full_name('Asabeneh', 'Yetayeh')) # Asabeneh Yetayeh


Nhập hàm từ 1 module

Chúng ta có thể có nhiều hàm trong một tệp và chúng ta có thể nhập tất cả các hàm theo nhiều cách khác nhau.
1. Ví dụ:
# main.py file from mymodule import generate_full_name, sum_two_nums, person, gravity print(generate_full_name('Asabneh','Yetayeh')) print(sum_two_nums(1,9)) mass = 100; weight = mass * gravity print(weight) print(person['firstname'])


Nhập hàm từ 1 module và đổi tên

Trong quá trình nhập, chúng ta có thể đổi tên
1. Ví dụ:
# main.py file from mymodule import generate_full_name as fullname, sum_two_nums as total, person as p, gravity as g print(fullname('Asabneh','Yetayeh')) print(total(1, 9)) mass = 100; weight = mass * g print(weight) print(p) print(p['firstname'])


Nhập built-in modules

Trong Python, bạn có thể sử dụng các module có sẵn để tận dụng các chức năng mà Python cung cấp không cần cài đặt thêm bất kỳ gói nào. Để sử dụng, bạn cần import chúng vào chương trình
1. Random Module:

Cung cấp cho chúng ta một số ngẫu nhiên trong khoảng từ 0 đến 0,9999... Một số chức năng:

from random import ramdom, randint print(random()) print(randint(5,20)) #trả về số nguyên trong khoảng [5,20] 2. Datetime Module: quản lý ngày giờ
from datetime import date date.today() #ngày hiện tại from datetime import time print(time(14,30)) #đại diện thời gian trong ngày (giờ, phút, giây, microseconds), không liên quan đến múi giờ from datetime import datetime datetime.now() #trả về ngày giờ, hiện tại datetime.strftime(format) #định dạng ngày giờ theo định dạng cụ thể from datetime import timedelta, datetime now = datetime.now() future_day = now + timedelta(days = 7) #tính toán ngày sau 7 ngày formatted_date = now.strftime("%Y-%m-%d, %H:%M:%S")

3. Math Module

Math module chứa nhiều phép toán, hằng số để thực hiện các phép toán khác nhau. Để kiểm tra những chức năng mà module có, chúng ta sử dụng help(math) hoặc dir(math). Điều này sẽ hiển thị các chức năng có sẵn trong module

import math print(math.sqrt(2)) #1.414.. from math import pi print(pi) from math import pi, sqrt, pow print(pi) #3.1415... print(sqrt(2)) #1.414... print(pow(2,3)) #8.0 from math import * from math import pi as PI print(PI)


PACKAGE

Package trong Python là một thư mục chứa một hoặc nhiều modules hay các package khác nhau, nó được tạo ra nhằm mục đích phân bố các modules có cùng chức năng hay một cái gì đó, để dễ quản lý source code.


Build package trong Python.

Để tạo ra một package trong python thì mọi người chỉ cần tạo ra một thư mục, với tên thư mục chính là tên của package và trong thư mục này nhất định phải có một file có tên __init__.py. File __init__.py này nó giống như các constructor, và nó sẽ được gọi ra đầu tiên khi chúng ta import package đó.
1. Cấu trúc:
|demopackage/__init__py |main.py # __init__.py print("Duoc in tu file __init__py") # main.py import demopackage

Sau đó các bạn chạy file main.py, sẽ thu được kết quả như sau: Duoc in tu file __init__py


Import package.

Để import các package thì các bạn sử dụng cú pháp giống như đối với modules. Nhưng bây giờ các bạn sẽ phải gọi từ package -> module.
1. Cấu trúc:
|--demopackage/ | |--modules.py | └── __init__py |--main.py # demopackage/modules.py def say_hello(): print("Hello!") def say_goodbye(): print("Goodbye!") # main.py from demopackage.modules import * # Duoc in tu file __init__py say_hello() # Hello! say_goodbye() # Goodbye! # main.py import demopackage.modules # Duoc in tu file __init__py demopackage.modules.say_hello() # Hello! demopackage.modules.say_goodbye() # Goodbye! # main.py import demopackage.modules as modules # Duoc in tu file __init__py modules.say_hello() # Hello! modules.say_goodbye() # Goodbye!


Package chứa package.

Đối với package trong Python thì nó cũng có thể chứa 1 hoặc nhiều package khác trong nó, và không dưới hạn số lần chứa đó nhưng các bạn vẫn phải luôn tuân thủ quy tắc của 1 package là phải có file __init__.py. Tức là package a chứa package b và package b lại chưa package c,...
1. Cấu trúc:
|--demopackage/ | |--modules.py | |--packagechild/ | |-- childmodule.py | └── __init__py | └── __init__py |--main.py # packagechild/childmodule.py def say(): print("childmodule.py: Hello") # packagechild/__init__.py print("childpackage __init__.py") # main.py from demopackage.packagechild.childmodule import * # childpackage __init__.py say() # childmodule.py: Hello


LIBRARIES

Trong Python, libraries là tập hợp các module chứa các hàm, lớp, và công cụ đã được viết sẵn để thực hiện các chức năng nhất định. Libraries giúp chúng ta tiết kiệm thời gian và công sức bằng cách cung cấp các chức năng tiện ích cho các tác vụ phổ biến như xử lý dữ liệu, tính toán khoa học, phân tích dữ liệu, v.v. Thay vì phải viết mã từ đầu, ta có thể sử dụng các hàm có sẵn từ thư viện.


Libraries Python Phổ Biến

Dưới đây là một số thư viện nổi bật trong Python, cùng với mô tả về cách sử dụng và ứng dụng của chúng:

  • math: Cung cấp các hàm toán học cơ bản như sqrt, pow, sin, cos, và factorial.
  • datetime: Làm việc với ngày và giờ.
  • random: Sinh số ngẫu nhiên, trộn danh sách ngẫu nhiên, chọn ngẫu nhiên phần tử.
  • os: Quản lý tệp và thư mục, truy cập hệ thống tệp.
  • sys: Xử lý các thông tin liên quan đến trình thông dịch Python, như đường dẫn hệ thống.
  • json và csv: Đọc và ghi tệp JSON, CSV.
  • collections: Các cấu trúc dữ liệu nâng cao như Counter, deque, namedtuple, và defaultdict.

FRAMEWORK

Framework trong Python là một bộ công cụ cung cấp cấu trúc sẵn có để phát triển các ứng dụng theo các nguyên tắc và kiến trúc tiêu chuẩn. Framework giúp lập trình viên tiết kiệm thời gian khi xây dựng ứng dụng bằng cách cung cấp các tính năng và thư viện đã được xây dựng và tối ưu hóa.


Các Framework Phổ Biến trong Python

Dưới đây là một số thư viện nổi bật trong Python, cùng với mô tả về cách sử dụng và ứng dụng của chúng:

1. Framework Web: Hỗ trợ phát triển ứng dụng web, xử lý yêu cầu HTTP, xác thực người dùng, quản lý cơ sở dữ liệu, và hiển thị nội dung.
  • Django: Django là một framework web mạnh mẽ và toàn diện. Nó tuân theo mô hình MVC (Model-View-Controller) và giúp dễ dàng phát triển ứng dụng lớn một cách nhanh chóng.
  • Flask: Flask là một micro-framework, nhẹ và linh hoạt. Thích hợp cho các dự án nhỏ và nhanh chóng.
  • FastAPI: Framework hiện đại để xây dựng API nhanh chóng, được tối ưu hóa cho hiệu suất cao và hỗ trợ khai báo kiểu dữ liệu (type hinting).
2. Framework Machine Learning
  • TensorFlow: Framework do Google phát triển, hỗ trợ học sâu (deep learning) và máy học (machine learning). Dùng trong xử lý ngôn ngữ tự nhiên, nhận dạng hình ảnh, phân tích dự báo, v.v.
  • PyTorch: Framework của Facebook, nổi bật với tính linh hoạt và dễ dàng sử dụng trong nghiên cứu.
3. Framework Phân Tích Dữ Liệu
  • TensorFlow: Apache Spark với PySpark
4. Framework Cho Các Ứng Dụng Desktop
  • Tkinter: Tkinter là framework tích hợp trong thư viện chuẩn Python để xây dựng GUI (giao diện người dùng đồ họa) đơn giản.
Ưu và Nhược Điểm của Framework
Ưu điểm
  • Tiết kiệm thời gian: Cung cấp nhiều tính năng sẵn có.
  • Bảo mật và ổn định: Được phát triển bởi các chuyên gia, giúp giảm thiểu lỗi bảo mật.
  • Tái sử dụng mã: Giúp mã dễ tái sử dụng, bảo trì và mở rộng.
  • Quy trình phát triển chuẩn hóa: Hỗ trợ các quy trình phát triển nhất quán.
Nhược điểm
  • Ít linh hoạt: Phải tuân theo cấu trúc và nguyên tắc của framework.
  • Độ phức tạp tăng: Với những ứng dụng đơn giản, framework có thể khiến dự án trở nên cồng kềnh.
  • Khả năng học tập: Đòi hỏi thời gian để làm quen với framework mới.

Module Location và thứ tự ưu tiên

Trong Python, Module location và thứ tự ưu tiên khi tìm kiếm module rất quan trọng để Python xác định đúng vị trí và thứ tự các module cần import. Cụ thể:


  • Thư mục hiện tại hoặc thư mục chứa script đang chạy: Python tìm kiếm tại thư mục hiện tại trước tiên. Điều này có nghĩa là nếu có một module cùng tên trong thư mục hiện tại, Python sẽ sử dụng module này thay vì tìm đến các thư mục khác.
  • PYTHONPATH: Nếu không tìm thấy module trong thư mục hiện tại, Python sẽ tìm tiếp trong các thư mục được chỉ định trong PYTHONPATH. Điều này giúp linh hoạt bổ sung các thư mục chứa module tùy chỉnh.
  • Thư viện chuẩn của Python: Nếu không có module trong thư mục hiện tại và PYTHONPATH, Python sẽ tìm trong thư viện chuẩn của Python (thư viện mặc định). Ví dụ, các module như math, datetime, và sys đều nằm trong thư viện chuẩn này.
  • Thư viện bên thứ ba: Python tìm tiếp các module được cài đặt từ bên thứ ba, chẳng hạn các module từ site-packages. Đây là nơi chứa các package được cài đặt qua pip.
1. Ví dụ
project_folder/ │ ├── my_module.py ├── script.py └── math.py # my_module.py def greet(): print("Hello from my_module!") # math.py def calculate(): print("This is a custom math module.") # script.py import math import my_module # Sử dụng module my_module my_module.greet() # Sử dụng module math math.calculate() 2. Thực thi script.py và kết quả mong đợi
  • 1. Python sẽ tìm module math trong thư mục hiện tại trước. Thay vì sử dụng module math chuẩn của Python, Python sẽ tìm thấy file math.py trong thư mục project_folder và import file này. Vì thế, khi gọi math.calculate(), Python sẽ gọi hàm calculate() trong file math.py của chúng ta thay vì module math của thư viện chuẩn.
  • 2. Module my_module: Python sẽ tìm thấy my_module.py trong cùng thư mục và import nó như mong đợi, vì không có module chuẩn nào tên là my_module.

* Lưu ý: Nếu bạn muốn sử dụng module math chuẩn của Python (cho các hàm như math.sqrt()), bạn nên đổi tên file math.py thành tên khác (ví dụ custom_math.py). Sau đó, bạn có thể import cả module chuẩn math và module tự tạo của mình


Giới thiệu về External Modules

External modules (các module bên ngoài) trong Python là những module và thư viện không nằm trong thư viện chuẩn của Python. Chúng được phát triển bởi cộng đồng hoặc các nhà phát triển bên ngoài, nhằm cung cấp các chức năng mở rộng và cụ thể cho nhiều lĩnh vực khác nhau. Việc sử dụng các external modules cho phép lập trình viên giải quyết những bài toán phức tạp mà không cần phải viết lại từ đầu, tiết kiệm thời gian và công sức.


1. Cài đặt External Modules

Để sử dụng các external modules, bạn thường sẽ cài đặt chúng thông qua trình quản lý package pip. Ví dụ, để cài đặt một module như requests, bạn có thể sử dụng lệnh sau trong terminal hoặc command prompt:

pip install requests 2. Một số ví dụ về External Modules

Để sử dụng các external modules, bạn thường sẽ cài đặt chúng thông qua trình quản lý package pip. Ví dụ, để cài đặt một module như requests, bạn có thể sử dụng lệnh sau trong terminal hoặc command prompt:

  • NumPy: Thư viện cho tính toán khoa học, cung cấp hỗ trợ cho mảng đa chiều và các phép toán số học.
  • Pandas: Dùng để phân tích dữ liệu, rất mạnh mẽ trong việc xử lý và phân tích dữ liệu dạng bảng.
  • Flask: Một micro framework để phát triển ứng dụng web nhẹ và nhanh
  • Django: Một framework mạnh mẽ cho phát triển ứng dụng web, cung cấp nhiều tính năng để xử lý cơ sở dữ liệu, quản lý phiên, xác thực người dùng, và nhiều hơn nữa.
  • TensorFlow: Thư viện cho học máy và AI, cung cấp công cụ cho việc xây dựng và triển khai các mô hình học sâu.

Cộng Đồng Phát Triển Python

Cộng đồng phát triển Python là một phần rất quan trọng giúp ngôn ngữ này trở nên phổ biến và mạnh mẽ. Dưới đây là một số đặc điểm và hoạt động nổi bật của cộng đồng này:

1. Diễn đàn và Nhóm Thảo luận
  • Stack Overflow: Một nơi tuyệt vời để hỏi và trả lời các câu hỏi liên quan đến Python.
  • Reddit: Có các subreddit như r/Python nơi người dùng chia sẻ kiến thức, bài viết, và hỏi đáp.
  • Mailing Lists: Python có nhiều mailing list cho các nhà phát triển để trao đổi ý kiến và giúp đỡ nhau.
2. Sự kiện và Hội nghị
  • PyCon: Một trong những hội nghị lớn nhất dành cho các lập trình viên Python. Đây là nơi để học hỏi, chia sẻ kinh nghiệm, và kết nối với những người khác trong cộng đồng.
  • Meetups: Các buổi gặp gỡ tại địa phương, nơi người dùng Python có thể chia sẻ kiến thức và kỹ năng của mình.
3. Tài liệu và Hướng dẫn
  • Documentation: Python có tài liệu chính thức rất đầy đủ, bao gồm cả hướng dẫn sử dụng cho các thư viện bên ngoài.
  • Tutorials: Có rất nhiều tutorial trực tuyến miễn phí và khóa học dành cho người mới bắt đầu và cả lập trình viên nâng cao.
4. Dự án mã nguồn mở

Nhiều dự án Python được phát triển dưới dạng mã nguồn mở, cho phép cộng đồng đóng góp, sửa lỗi, và cải thiện chất lượng của các module và thư viện. GitHub là nơi phổ biến nhất để lưu trữ và cộng tác trên các dự án này.

5. Tính linh hoạt và Đổi mới

Cộng đồng Python rất cởi mở và linh hoạt trong việc chấp nhận các ý tưởng mới. Điều này dẫn đến việc phát triển nhiều thư viện và công cụ mới, giúp Python luôn cập nhật và phù hợp với nhu cầu hiện tại.
