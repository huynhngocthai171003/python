MODULE TRONG PYTHON

Trong Python, module và package là hai khái niệm cơ bản giúp tổ chức và quản lý mã nguồn một cách hiệu quả.

Module

Một module trong Python là một file chứa định nghĩa và câu lệnh Python. File này có phần mở rộng là .py. Module có thể chứa định nghĩa của hàm, lớp và biến, cũng như mã thực thi. Mục đích chính của việc sử dụng module là để tái sử dụng mã nguồn. Bạn có thể định nghĩa các hàm, lớp và biến trong một module và sau đó sử dụng chúng trong một chương trình khác thông qua việc import module đó.

Package

Package trong Python là một thư mục chứa một hoặc nhiều modules hay các package khác nhau, nó được tạo ra nhằm mục đích phân bố các modules có cùng chức năng hay một cái gì đó, để dễ quản lý source code.


Tạo 1 module

Để tạo 1 module, chúng ta viết mã bằng tập lệnh Python và lưu nó dưới dạng tệp .py. Tạo 1 một tệp có tên mymodule.py trong thư mục dự án.
1. Ví dụ: #mymodule.py def generate_full_name(firstname, lastname): space = ' ' fullname = firstname + space + lastname return fullname def sum_two_nums (num1, num2): return num1 + num2 gravity = 9.81 person = { "firstname": "Asabeneh", "age": 250, "country": "Finland", "city":'Helsinki' }


Nhập vào 1 module

Để nhập tệp, chúng ta sử dụng từ khóa import và tên của tệp
1. Ví dụ:
#main.py import mymodule print(mymodule.generate_full_name('Asabeneh', 'Yetayeh')) # Asabeneh Yetayeh


Nhập hàm từ 1 module

Chúng ta có thể có nhiều hàm trong một tệp và chúng ta có thể nhập tất cả các hàm theo nhiều cách khác nhau.
1. Ví dụ:
# main.py file from mymodule import generate_full_name, sum_two_nums, person, gravity print(generate_full_name('Asabneh','Yetayeh')) print(sum_two_nums(1,9)) mass = 100; weight = mass * gravity print(weight) print(person['firstname'])


Nhập hàm từ 1 module và đổi tên

Trong quá trình nhập, chúng ta có thể đổi tên
1. Ví dụ:
# main.py file from mymodule import generate_full_name as fullname, sum_two_nums as total, person as p, gravity as g print(fullname('Asabneh','Yetayeh')) print(total(1, 9)) mass = 100; weight = mass * g print(weight) print(p) print(p['firstname'])


Nhập built-in modules

Trong Python, bạn có thể sử dụng các module có sẵn để tận dụng các chức năng mà Python cung cấp không cần cài đặt thêm bất kỳ gói nào. Để sử dụng, bạn cần import chúng vào chương trình
1. Random Module:

Cung cấp cho chúng ta một số ngẫu nhiên trong khoảng từ 0 đến 0,9999... Một số chức năng:

from random import ramdom, randint print(random()) print(randint(5,20)) #trả về số nguyên trong khoảng [5,20] 2. Datetime Module: quản lý ngày giờ
from datetime import date date.today() #ngày hiện tại from datetime import time print(time(14,30)) #đại diện thời gian trong ngày (giờ, phút, giây, microseconds), không liên quan đến múi giờ from datetime import datetime datetime.now() #trả về ngày giờ, hiện tại datetime.strftime(format) #định dạng ngày giờ theo định dạng cụ thể from datetime import timedelta, datetime now = datetime.now() future_day = now + timedelta(days = 7) #tính toán ngày sau 7 ngày formatted_date = now.strftime("%Y-%m-%d, %H:%M:%S")

3. Math Module

Math module chứa nhiều phép toán, hằng số để thực hiện các phép toán khác nhau. Để kiểm tra những chức năng mà module có, chúng ta sử dụng help(math) hoặc dir(math). Điều này sẽ hiển thị các chức năng có sẵn trong module

import math print(math.sqrt(2)) #1.414.. from math import pi print(pi) from math import pi, sqrt, pow print(pi) #3.1415... print(sqrt(2)) #1.414... print(pow(2,3)) #8.0 from math import * from math import pi as PI print(PI)
