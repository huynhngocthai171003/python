GUI - Picture

Đối tượng Picture dùng để hiển thị một hình ảnh trong ứng dụng.


Cách sử dụng Picture trên Windows:

Cú pháp:
from guizero import App, Picture app = App() picture = Picture(app, image="test.gif") app.display() Bạn phải chỉ định đúng đường dẫn của hình ảnh. Trong ví dụ trên, hình ảnh nằm trong cùng thư mục với chương trình. Nếu hình ảnh nằm trong thư mục khác, bạn cần chỉ định đường dẫn tương đối. Ví dụ, nếu hình ảnh nằm trong một thư mục con có tên là "images", bạn có thể viết như sau: picture = Picture(app, image="images/test.gif") Hỗ trợ hình ảnh phụ thuộc vào hệ điều hành của bạn và việc cài đặt các tính năng hình ảnh bổ sung của guizero. Theo mặc định, GIF và PNG được hỗ trợ, trừ macOS chỉ hỗ trợ GIF.

Tham số khởi tạo

Khi tạo một đối tượng Picture, bạn cần chỉ định tham số master và có thể sử dụng các tham số tùy chọn. Các tham số này được chỉ định trong dấu ngoặc, ví dụ:

Tham số Kiểu dữ liệu Mặc định Bắt buộc Mô tả
master App, Window, Box - Có Container chứa widget này
image string None Không Đường dẫn đến tệp hình ảnh (hoặc đối tượng tkinter.PhotoImage hoặc PIL.Image)
align string None Không Căn chỉnh widget trong container. Các giá trị có thể: "top", "bottom", "left", "right"
grid List [int, int] None Không Tọa độ [x, y] của widget. Cần thiết nếu container có layout dạng grid
visible boolean True Không Xác định xem widget có hiển thị hay không
enabled boolean None Không Xác định widget có được kích hoạt hay không
width size None Không Đặt chiều rộng của widget (theo đơn vị pixel)
height size None Không Đặt chiều cao của widget (theo đơn vị pixel)

Ví dụ:

# Tạo một ứng dụng với các tham số như màu nền, chiều rộng, chiều cao và tiêu đề app = App(bg="red", height=200, width=300, title="Ứng dụng của tôi") # Cấu hình thêm các tham số khác app.image = "path/to/image.png" # Đặt hình ảnh app.align = "top" # Căn chỉnh widget app.grid = [0, 1] # Đặt tọa độ trong layout grid app.visible = False # Ẩn cửa sổ app.enabled = True # Kích hoạt widget app.display() # Hiển thị ứng dụng

Các phương thức của Picture

Bạn có thể gọi các phương thức sau cho đối tượng Picture:

Phương thức Tham số Trả về Mô tả
after time (int), command (function name), args (list) - Lên lịch gọi một hàm sau thời gian time (milliseconds).
cancel command (function name) - Hủy bỏ một cuộc gọi đã lên lịch.
destroy - - Hủy bỏ widget.
disable - - Vô hiệu hóa widget, khiến widget không thể tương tác.
enable - - Kích hoạt widget.
focus - - Đặt widget làm đối tượng có tiêu điểm.
hide - - Ẩn widget khỏi giao diện.
repeat time (int), command (function name), args (list) - Lặp lại việc gọi một hàm sau mỗi thời gian time (milliseconds).
resize width (int), height (int) - Đặt lại chiều rộng và chiều cao của widget.
show - - Hiển thị widget nếu nó bị ẩn trước đó.

1. after: Lên lịch gọi một hàm sau thời gian nhất định.

from guizero import App def say_hello(): print("Hello after 2 seconds!") app = App("Example: after") app.after(2000, say_hello) # Gọi hàm say_hello sau 2 giây app.display()

2. cancel: Hủy bỏ lệnh đã lên lịch.

from guizero import App def say_hello(): print("This will not be printed.") app = App("Example: cancel") timer = app.after(2000, say_hello) # Lên lịch gọi hàm say_hello sau 2 giây app.cancel(timer) # Hủy bỏ lệnh trên app.display()

3. destroy: Hủy bỏ widget.

from guizero import App, Box, PushButton def on_click(): button.destroy() # Hủy bỏ widget app = App("Example: destroy") button = PushButton(app, text="Click to destroy", command=on_click) app.display()

4. disable: Vô hiệu hóa widget, khiến widget không thể tương tác.

from guizero import App, PushButton app = App("Example: disable") button = PushButton(app, text="I am disabled") button.disable() # Vô hiệu hóa widget app.display()

5. enable: Kích hoạt widget.

from guizero import App, PushButton app = App("Example: enable") button = PushButton(app, text="I am enabled") button.disable() # Vô hiệu hóa widget button.enable() # Kích hoạt lại widget app.display()

6. focus: Đặt widget làm đối tượng có tiêu điểm.

from guizero import App, TextBox app = App("Example: focus") textbox = TextBox(app) textbox.focus() # Đặt TextBox làm đối tượng có tiêu điểm app.display()

7. hide: Ẩn widget khỏi giao diện.

from guizero import App, PushButton app = App("Example: hide") button = PushButton(app, text="I am hidden") button.hide() # Ẩn widget khỏi giao diện app.display()

8. repeat: Lặp lại việc gọi một hàm sau mỗi thời gian time (milliseconds).

from guizero import App def say_hello(): print("Hello!") app = App("Example: repeat") app.repeat(2000, say_hello) # Lặp lại việc gọi say_hello sau mỗi 2 giây app.display()

9. resize: Đặt lại chiều rộng và chiều cao của widget.

from guizero import App, Box app = App("Example: resize") box = Box(app, width=100, height=100) box.resize(200, 200) # Đặt lại kích thước widget Box app.display()

10. focus: Đặt tiêu điểm cho ứng dụng.

from guizero import App app = App("Example: focus") app.focus() # Đặt tiêu điểm vào ứng dụng app.display()

10. show: Hiển thị widget nếu nó bị ẩn trước đó.

from guizero import App, PushButton app = App("Example: show") button = PushButton(app, text="I am hidden") button.hide() # Ẩn widget button.show() # Hiển thị lại widget app.display()

Các thuộc tính Picture

Bạn có thể thiết lập và lấy các giá trị của các thuộc tính sau của đối tượng Picture:

Thuộc tính Kiểu dữ liệu Mô tả
align string Căn chỉnh widget trong container
bg color Màu nền của widget
enabled boolean True nếu widget đang được kích hoạt
grid List Tọa độ [x, y] của widget, chỉ cần thiết khi master có layout dạng grid
height size Chiều cao của widget (theo đơn vị pixel)
image string Đường dẫn đến hình ảnh bạn muốn hiển thị
master App, Box Container chứa widget này
tk tkinter.Label Đối tượng tkinter nội bộ, xem trong phần "Sử dụng tkinter"
value string Đường dẫn đến hình ảnh bạn muốn hiển thị
visible boolean Nếu widget có hiển thị
width size Chiều rộng của widget (theo đơn vị pixel)

1. align: Căn chỉnh hình ảnh trong widget.

from guizero import Picture picture = Picture(app, align="top", image="example.png") picture.display()

2. bg: Màu nền của widget.

from guizero import Picture picture = Picture(app, bg="lightblue", image="example.png") picture.display()

3. enabled: True nếu widget đang được kích hoạt.

from guizero import Picture picture = Picture(app, enabled=True, image="example.png") picture.display()

4. height: Chiều cao của widget (theo đơn vị pixel).

from guizero import Picture picture = Picture(app, height=100, image="example.png") picture.display()

5. image: Đường dẫn đến hình ảnh bạn muốn hiển thị.

from guizero import Picture picture = Picture(app, image="example.png") picture.display()

6. master: Container chứa widget này.

from guizero import App, Picture app = App("Example") picture = Picture(app, master=app, image="example.png") picture.display()

7. value: Đường dẫn đến hình ảnh bạn muốn hiển thị.

from guizero import Picture picture = Picture(app, value="example.png") picture.display()

8. visible: Nếu widget có hiển thị.

from guizero import Picture picture = Picture(app, visible=True, image="example.png") picture.display()

9. width: Chiều rộng của widget (theo đơn vị pixel).

from guizero import Picture picture = Picture(app, width=100, image="example.png") picture.display()

10. grid: Tọa độ [x, y] của widget nếu sử dụng lưới trong container.

from guizero import Picture picture = Picture(app, grid=[0, 1], image="example.png") picture.display()

11. tk: Đối tượng tkinter.Label nội bộ.

from guizero import Picture picture = Picture(app, tk=None, image="example.png") picture.display()
