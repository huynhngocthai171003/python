Các Chế Độ Mở File (File Modes)

Chế độ (mode) Ý nghĩa Hành động Ghi chú Ví dụ
"r" Read (Chỉ đọc) Mở file để đọc, lỗi nếu file không tồn tại Mặc định khi dùng open() with open("file.txt", "r") as f: data = f.read()
"w" Write (Ghi mới) Ghi đè nội dung, tạo file nếu chưa tồn tại Xóa toàn bộ nội dung cũ with open("file.txt", "w") as f: f.write("Hello!")
"a" Append (Ghi thêm) Thêm nội dung vào cuối file, tạo file nếu chưa có Không xóa nội dung cũ with open("file.txt", "a") as f: f.write("\nNew line")
"x" Exclusive create Tạo file mới, lỗi nếu file đã tồn tại Dùng để tránh ghi đè file quan trọng with open("file.txt", "x") as f: f.write("New file")
"rb" Read binary Đọc file ở chế độ nhị phân (hình ảnh, video, v.v.) Không cần encoding="utf-8" with open("image.jpg", "rb") as f: data = f.read()
"wb" Write binary Ghi đè file nhị phân, tạo file nếu chưa có Thường dùng cho ảnh, âm thanh, video with open("image.jpg", "wb") as f: f.write(binary_data)
"ab" Append binary Ghi thêm vào file nhị phân Giữ lại nội dung cũ with open("sound.mp3", "ab") as f: f.write(more_data)
Sử dụng with open(...) trong Python có nhiều lợi ích so với việc chỉ dùng open(...), đặc biệt là khi làm việc với tệp tin.
  1. Tự động đóng file sau khi hoàn thành
    • Khi dùng with open(...), Python tự động đóng file sau khi thoát khỏi khối lệnh with, ngay cả khi có lỗi xảy ra.
    • Điều này giúp tránh rò rỉ tài nguyên và giảm nguy cơ lỗi khi quên gọi close().
  2. Mã ngắn gọn, dễ đọc
    • Không cần gọi f.close() thủ công.
    • Giúp code gọn hơn và ít lỗi hơn.
  3. Bảo vệ khỏi lỗi khi đọc/ghi file
    • Nếu có lỗi xảy ra trong quá trình đọc hoặc ghi file, Python vẫn đảm bảo file được đóng đúng cách.
Không dùng with (có thể gây lỗi nếu quên close())
f = open("file.txt", "r") data = f.read() f.close() # Nếu quên gọi dòng này, file sẽ không được đóng Dùng with open(...) (an toàn hơn)
with open("file.txt", "r") as f: data = f.read() # File tự động đóng khi thoát khỏi khối with

Hàm mở file

Hàm Chức năng Ví dụ
open("file.txt", "r") Mở file để đọc file = open("data.txt", "r")
open("file.txt", "w") Mở file để ghi (xóa nội dung cũ) file = open("data.txt", "w")
open("file.txt", "a") Mở file để ghi thêm file = open("data.txt", "a")
with open("file.txt", "r") as f Mở file với cú pháp with open() Tránh quên file.close()
with open("data.txt", "r", encoding="utf-8") as file: content = file.read() print(content) # In nội dung file

Đọc File

Hàm Chức năng Ví dụ
file.read() Đọc toàn bộ nội dung file content = file.read()
file.readline() Đọc 1 dòng từ file line = file.readline()
file.readlines() Đọc tất cả các dòng, trả về danh sách lines = file.readlines()
for line in file Đọc từng dòng trong vòng lặp for line in file: print(line.strip())
with open("data.txt", "r", encoding="utf-8") as file: for line in file: print(line.strip()) # Loại bỏ khoảng trắng và in ra từng dòng

Ghi File

Hàm Chức năng Ví dụ
file.write("text") Ghi nội dung vào file (ghi đè) file.write("Xin chào!")
file.writelines([...]) Ghi danh sách dòng vào file file.writelines(["Dòng 1\n", "Dòng 2\n"])
Ví dụ ghi file:
with open("data.txt", "w", encoding="utf-8") as file: file.write("Xin chào! Đây là dòng đầu tiên.\n") file.write("Học lập trình Python thật thú vị!\n") Nếu mở file với "w", nội dung cũ sẽ bị xóa.
Ví dụ ghi thêm vào file:
with open("data.txt", "a", encoding="utf-8") as file: file.write("Thêm dòng mới vào cuối file.\n") Mở file với "a" để giữ nguyên nội dung cũ và chỉ thêm vào cuối.

Đóng File

Hàm Chức năng Ví dụ
file.close() Đóng file sau khi đọc hoặc ghi file.close()
Ví dụ đóng file thủ công:
file = open("data.txt", "r", encoding="utf-8") content = file.read() file.close() # Đóng file để giải phóng tài nguyên Sử dụng with open() để tự động đóng file (khuyên dùng):
with open("data.txt", "r", encoding="utf-8") as file: content = file.read() Ưu điểm: Không cần gọi file.close(), giúp tránh lỗi quên đóng file.
Ví dụ tổng quát
# Bước 1: Mở file để ghi với UTF-8 with open("example.txt", "w", encoding="utf-8") as file: file.write("Xin chào! Đây là dòng đầu tiên.\n") file.write("Học lập trình Python thật thú vị!\n") # Bước 2: Mở file để đọc với UTF-8 with open("example.txt", "r", encoding="utf-8") as file: content = file.read() print("Nội dung trong file:") print(content) # Bước 3: Mở file để ghi thêm nội dung mới với UTF-8 with open("example.txt", "a", encoding="utf-8") as file: file.write("Chúng ta có thể thêm nội dung vào file mà không mất dữ liệu cũ.\n") # Bước 4: Đọc lại file sau khi thêm nội dung với UTF-8 with open("example.txt", "r", encoding="utf-8") as file: print("Nội dung sau khi cập nhật:") print(file.read())

Xử lý ngoại lệ với try-except trong Python kết hợp với guizero

Trong Python, try-except được sử dụng để xử lý ngoại lệ (lỗi) khi chạy chương trình. Khi kết hợp với thư viện guizero, ta có thể hiển thị thông báo lỗi trực tiếp trên giao diện thay vì chỉ in ra màn hình console.

Cấu trúc try-except cơ bản trong Python

try: # Đoạn code có thể gây lỗi x = int(input("Nhập một số: ")) print(f"Số bạn vừa nhập là: {x}") except ValueError: # Xử lý khi xảy ra lỗi print("Lỗi: Bạn phải nhập một số nguyên!")

Kết hợp try-except với guizero

Ví dụ 1: Bắt lỗi nhập sai dữ liệu trong guizero
from guizero import App, TextBox, PushButton, Text, info, error def check_number(): try: num = int(input_box.value) # Chuyển đổi giá trị nhập vào thành số nguyên info("Kết quả", f"Số bạn vừa nhập là: {num}") # Hiển thị thông báo thành công except ValueError: error("Lỗi", "Vui lòng nhập một số nguyên!") # Hiển thị hộp thoại lỗi # Tạo giao diện app = App(title="Nhập số", width=300, height=200) Text(app, text="Nhập một số nguyên:") input_box = TextBox(app) button = PushButton(app, text="Kiểm tra", command=check_number) app.display() Ví dụ 2: Xử lý lỗi khi mở file trong guizero
from guizero import App, Text, TextBox, PushButton, error def open_file(): try: with open(filename_box.value, "r", encoding="utf-8") as file: content = file.read() result.value = content # Hiển thị nội dung file except FileNotFoundError: error("Lỗi", "File không tồn tại! Vui lòng nhập lại.") except Exception as e: error("Lỗi", f"Lỗi khác: {str(e)}") # Giao diện chính app = App(title="Đọc file", width=400, height=300) Text(app, text="Nhập tên file cần mở:") filename_box = TextBox(app) button = PushButton(app, text="Mở file", command=open_file) result = Text(app, text="") app.display()
Loại lỗi Mô tả Ví dụ
Exception Lớp cha của mọi lỗi trong Python. try: 1/0 except Exception as e: print(e)
SyntaxError Lỗi cú pháp khi code sai. print("Hello" (Thiếu dấu ngoặc đóng)
IndentationError Lỗi thụt lề không đúng. def func(): (Thiếu thụt lề)
NameError Biến chưa được định nghĩa. print(x) (x chưa khai báo)
TypeError Dùng sai kiểu dữ liệu trong phép toán. "3" + 5 (Cộng chuỗi với số)
ValueError Giá trị sai nhưng đúng kiểu dữ liệu. int("abc") (Không thể ép "abc" thành số)
IndexError Truy cập phần tử vượt quá danh sách. lst = [1,2,3]; print(lst[5])
KeyError Truy cập key không tồn tại trong dictionary. d = {"a": 1}; print(d["b"])
FileNotFoundError Cố mở file không tồn tại. open("no_file.txt")
ZeroDivisionError Chia số cho 0. 5 / 0
AttributeError Thuộc tính không tồn tại trong object. None.length (None không có `length`)
