GUI - Thư viện OS

Thư viện os trong Python dùng để làm việc với hệ điều hành, cho phép quản lý file, thư mục, biến môi trường, và nhiều thao tác khác.


1. Quản lý thư mục

Tạo thư mục

  • Hàm: os.mkdir(path)
  • Mô tả: Tạo một thư mục mới tại đường dẫn chỉ định.
import os os.mkdir("new_directory") # Tạo thư mục mới có tên "new_directory"

Tạo nhiều thư mục lồng nhau

  • Hàm: os.makedirs(path)
  • Mô tả: Tạo nhiều thư mục lồng nhau.
os.makedirs("parent_directory/child_directory") # Tạo thư mục cha và thư mục con

Xóa thư mục

  • Hàm: os.rmdir(path)
  • Mô tả: Xóa một thư mục rỗng.
os.rmdir("new_directory") # Xóa thư mục "new_directory" nếu nó rỗng

Xóa nhiều thư mục lồng nhau

  • Hàm: os.removedirs(path)
  • Mô tả: Xóa nhiều thư mục lồng nhau.
os.removedirs("parent_directory/child_directory") # Xóa thư mục con và thư mục cha nếu rỗng

Thay đổi thư mục làm việc

  • Hàm: os.chdir(path)
  • Mô tả: Thay đổi thư mục làm việc hiện tại.
os.chdir("new_directory") # Thay đổi thư mục làm việc hiện tại sang "new_directory"

Lấy đường dẫn thư mục làm việc hiện tại

  • Hàm: os.getcwd()
  • Mô tả: Lấy đường dẫn thư mục làm việc hiện tại.
current_directory = os.getcwd() print(current_directory) # In ra đường dẫn thư mục làm việc hiện tại

2. Quản lý tệp

Liệt kê tệp trong thư mục

  • Hàm: os.listdir(path)
  • Mô tả: Liệt kê các tệp và thư mục trong một thư mục.
files = os.listdir(".") # Liệt kê các tệp trong thư mục hiện tại print(files)

Xóa tệp

  • Hàm: os.remove(path)
  • Mô tả: Xóa một tệp.
os.remove("file.txt") # Xóa tệp "file.txt"

Đổi tên tệp hoặc thư mục

  • Hàm: os.rename(src, dst)
  • Mô tả: Đổi tên một tệp hoặc thư mục.
os.rename("old_name.txt", "new_name.txt") # Đổi tên tệp từ "old_name.txt" sang "new_name.txt"

3. Thông tin hệ thống

Truy cập biến môi trường

  • Biến: os.environ
  • Mô tả: Truy cập và thay đổi biến môi trường.
print(os.environ['HOME']) # In ra đường dẫn thư mục chính của người dùng

4. Làm việc với đường dẫn

Thư viện os.path cung cấp các hàm hữu ích để làm việc với đường dẫn tệp và thư mục.

os.path.join()

  • Mô tả: Kết hợp các phần của đường dẫn thành một đường dẫn hợp lệ.
  • Cú pháp: os.path.join(path, *paths)
import os path = os.path.join("folder", "subfolder", "file.txt") print(path) # Kết quả có thể là "folder/subfolder/file.txt" trên Unix hoặc "folder\subfolder\file.txt" trên Windows

os.path.exists()

  • Mô tả: Kiểm tra xem một đường dẫn có tồn tại hay không.
  • Cú pháp: os.path.exists(path)
import os if os.path.exists("file.txt"): print("Tệp tồn tại.") else: print("Tệp không tồn tại.")

os.path.isfile()

  • Mô tả: Kiểm tra xem một đường dẫn có phải là tệp hay không.
  • Cú pháp: os.path.isfile(path)
import os if os.path.isfile("file.txt"): print("Đây là một tệp.") else: print("Đây không phải là một tệp.")

os.path.isdir()

  • Mô tả: Kiểm tra xem một đường dẫn có phải là thư mục hay không.
  • Cú pháp: os.path.isdir(path)
import os if os.path.isdir("folder"): print("Đây là một thư mục.") else: print("Đây không phải là một thư mục.")

os.path.abspath()

  • Mô tả: Trả về đường dẫn tuyệt đối của một đường dẫn tương đối.
  • Cú pháp: os.path.abspath(path)
import os relative_path = "file.txt" absolute_path = os.path.abspath(relative_path) print("Đường dẫn tuyệt đối:", absolute_path)

os.path.basename()

  • Mô tả: Trả về tên tệp hoặc thư mục từ một đường dẫn.
  • Cú pháp: os.path.basename(path)
import os path = "/home/user/file.txt" filename = os.path.basename(path) print("Tên tệp:", filename) # Kết quả: "file.txt"

os.path.dirname()

  • Mô tả: Trả về thư mục chứa tệp từ một đường dẫn.
  • Cú pháp: os.path.dirname(path)
import os path = "/home/user/file.txt" directory = os.path.dirname(path) pint("Thư mục chứa:", directory) # Kết quả: "/home/user"

os.path.split()

  • Mô tả: Tách đường dẫn thành thư mục và tên tệp.
  • Cú pháp: os.path.split(path)
import os path = "/home/user/file.txt" directory, filename = os.path.split(path) print("Thư mục:", directory) # Kết quả: "/home/user" print("Tên tệp:", filename) # Kết quả: "file.txt"

os.path.splitext()

  • Mô tả: Tách tên tệp thành tên và phần mở rộng.
  • Cú pháp: os.path.splitext(path)
import os path = "example.txt" name, extension = os.path.splitext(path) print("Tên tệp:", name) # Kết quả: "example" print("Phần mở rộng:", extension) # Kết quả: ".txt"

os.path.splitext()

  • Mô tả: Tách tên tệp thành tên và phần mở rộng.
  • Cú pháp: os.path.splitext(path)
import os path = "example.txt" name, extension = os.path.splitext(path) print("Tên tệp:", name) # Kết quả: "example" print("Phần mở rộng:", extension) # Kết quả: ".txt"
