GUI - Text

Text Widget


Cách tạo một Text Widget

Để tạo một Text widget, bạn cần sử dụng lớp Text và chỉ định một ứng dụng (App) nơi mà văn bản sẽ hiển thị. Bạn có thể cấu hình văn bản theo nhiều cách khác nhau, bao gồm việc thay đổi màu sắc, font chữ, kích thước, căn chỉnh, v.v.

Cú pháp:
from guizero import App, Text app = App("Chương trình của tôi") text = Text(app, text="Chào mừng bạn đến với guizero!") app.display()

Các tham số của Text Widget

Khi bạn tạo một đối tượng Văn bản, bạn phải chỉ định một đối tượng chính và bạn có thể chỉ định bất kỳ tham số tùy chọn nào. Chỉ định các tham số trong ngoặc, như thế này: text = Text(app, text="hi")

Tham số Kiểu dữ liệu Mặc định Bắt buộc Mô tả
master App, Window hoặc Box - Có Đối tượng chứa widget này.
align Chuỗi None Không Định vị trí widget trong container. Các giá trị có thể: "top", "bottom", "left", "right".
color Màu black Không Màu sắc của văn bản, có thể là tên màu như "red" hoặc mã màu hex như #0099ff.
font Chuỗi "Helvetica" Không Phông chữ của văn bản.
grid Danh sách [int, int] None Không Tọa độ [x, y] của widget nếu container sử dụng bố cục "grid".
size Số nguyên 12 Không Kích thước phông chữ của văn bản.
text Chuỗi "" Không Văn bản sẽ hiển thị trong widget.
visible Boolean True Không Chỉ ra liệu widget có hiển thị hay không.
enabled Boolean None Không Chỉ ra liệu widget có được kích hoạt hay không.
width Số nguyên None Không Chiều rộng của widget tính theo ký tự hoặc "fill".
height Số nguyên None Không Chiều cao của widget tính theo ký tự hoặc "fill".
bold Boolean None Không Đặt văn bản thành in đậm.
italic Boolean None Không Đặt văn bản thành in nghiêng.
underline Boolean None Không Đặt văn bản có gạch chân.
overstrike Boolean None Không Đặt văn bản có gạch ngang.

Ví dụ về master:

from guizero import App, Text app = App() my_text = Text(app, text="Hello, master widget", size=12) app.display()

Ví dụ về align:

my_text = Text(app, text="Aligned Text", align="center")

Ví dụ về color:

my_text = Text(app, text="Red Text", color="red")

Ví dụ về font:

my_text = Text(app, text="Custom Font Text", font="Arial")

Ví dụ về grid:

my_text = Text(app, text="Grid Position", grid=[0, 0])

Ví dụ về size:

my_text = Text(app, text="Large Text", size=20)

Ví dụ về text:

my_text = Text(app, text="This is the text content.")

Ví dụ về visible:

my_text = Text(app, text="Visible Text", visible=True)

Ví dụ về enabled:

my_text = Text(app, text="This text is enabled", enabled=True)

Ví dụ về width:

my_text = Text(app, text="Wide Text", width=15)

Ví dụ về height:

my_text = Text(app, text="Tall Text", height=5)

Ví dụ về bold:

my_text = Text(app, text="Bold Text", bold=True)

Ví dụ về italic:

my_text = Text(app, text="Italic Text", italic=True)

Ví dụ về underline:

my_text = Text(app, text="Underlined Text", underline=True)

Ví dụ về overstrike:

my_text = Text(app, text="Struck Through Text", overstrike=True)

Các phương thức của Text

ABạn có thể gọi các phương thức sau trên đối tượng Text phổ biến:

Method Takes Returns Description
after time (int), command (function name), args (list) - Lên lịch gọi một lệnh sau một khoảng thời gian (mili giây).
append text (string) - Thêm văn bản vào cuối văn bản hiện tại.
cancel command (function name) - Hủy một lệnh đã lên lịch.
clear - - Xóa văn bản hiện tại.
destroy - - Hủy widget.
disable - - Vô hiệu hóa widget.
enable - - Kích hoạt lại widget.
focus - - Đặt focus vào widget.
hide - - Ẩn widget khỏi màn hình.
repeat time (int), command (function name), args (list) - Lặp lại lệnh sau mỗi khoảng thời gian.
resize width (int), height (int) - Đặt chiều rộng và chiều cao của widget.
show - - Hiển thị widget nếu trước đó bị ẩn.

1. after: Lên lịch gọi một lệnh sau một khoảng thời gian.

from guizero import App, Text def say_hello(): text.value = "Hello!" app = App("Example: after") text = Text(app, text="Waiting...") app.after(2000, say_hello) # Executes say_hello after 2 seconds app.display()

2. append: Thêm văn bản vào cuối văn bản hiện tại.

from guizero import App, Text app = App("Example: append") text = Text(app, text="Initial Text") text.append(" and more text.") # Adds to the existing text app.display()

3. cancel: Hủy một lệnh đã lên lịch.

from guizero import App, Text def say_hello(): text.value = "Hello!" app = App("Example: cancel") text = Text(app, text="Waiting...") call = app.after(2000, say_hello) # Schedule say_hello app.cancel(call) # Cancels the scheduled call app.display()

4. clear: Xóa văn bản hiện tại.

from guizero import App, Text app = App("Example: clear") text = Text(app, text="Some text") text.clear() # Clears the text app.display()

5. destroy: Hủy widget.

from guizero import App, Text app = App("Example: destroy") text = Text(app, text="This widget will be destroyed") text.destroy() # Destroys the widget app.display()

6. disable: Vô hiệu hóa widget.

from guizero import App, PushButton def on_click(): print("Button clicked!") app = App("Example: disable") button = PushButton(app, text="Click Me", command=on_click) button.disable() # Disables the button app.display()

7. enable: Kích hoạt lại widget.

from guizero import App, PushButton def on_click(): print("Button clicked!") app = App("Example: enable") button = PushButton(app, text="Click Me", command=on_click) button.disable() # Disable the button button.enable() # Enables the button back app.display()

8. focus: Đặt focus vào widget.

from guizero import App, TextBox app = App("Example: focus") textbox = TextBox(app) textbox.focus() # Sets focus on the TextBox app.display()

9. hide: Ẩn widget khỏi màn hình.

from guizero import App, Text app = App("Example: hide") text = Text(app, text="This text will be hidden") text.hide() # Hides the text widget app.display()

10. repeat: Lặp lại lệnh sau mỗi khoảng thời gian.

from guizero import App, Text def update_text(): text.value = "Updated!" app = App("Example: repeat") text = Text(app, text="Initial Text") app.repeat(2000, update_text) # Repeats update_text every 2 seconds app.display()

11. resize: Đặt chiều rộng và chiều cao của widget.

from guizero import App, Box app = App("Example: resize") box = Box(app) box.resize(200, 100) # Resizes the box to 200x100 app.display()

12. show: Hiển thị widget nếu trước đó bị ẩn.

from guizero import App, Text app = App("Example: show") text = Text(app, text="This text is initially hidden") text.hide() # Hides the text text.show() # Shows the text again app.display()

Các thuộc tính Text

Bạn có thể đặt và nhận các thuộc tính sau:

Method Data type Description
align string The alignment of this widget within its container
bg color The background colour of the widget
enabled boolean True if the widget is enabled
font string The font of the text
grid List [x,y] coordinates of this widget. This parameter is only required if the master object has a grid
height size Set the height of the widget in characters or to "fill"
master App or Box The container to which this widget belongs
text_size int The size of the text
text_color color The colour of the text
text_bold boolean Whether the text is bold
text_italic boolean Whether the text is italic
text_underline boolean Whether the text is underlined
text_overstrike boolean Whether the text is overstruck
tk tkinter.Label The internal tkinter object, see Using tkinter
value string The text
visible boolean If this widget is visible
width size Set the width of the widget in characters or to "fill"

1. align: Căn chỉnh của widget trong container.

from guizero import App, Text app = App("Example: align") message = Text(app, text="Hello, World!", align="center") app.display()

2. bg: Màu nền của widget.

from guizero import App app = App("Example: bg", bg="lightblue") app.display()

3. enabled: True nếu widget được kích hoạt.

from guizero import App, Text app = App("Example: enabled") message = Text(app, text="Hello, World!", enabled=True) app.display()

4. font: Phông chữ của văn bản.

from guizero import App, Text app = App("Example: font") message = Text(app, text="Hello, World!", font="Arial") app.display()

5. grid: Tọa độ [x, y] của widget.

from guizero import App, Text app = App("Example: grid") message = Text(app, text="Hello, World!", grid=[0, 0]) app.display()

6. height: Chiều cao của widget.

from guizero import App, Text app = App("Example: height") message = Text(app, text="Hello, World!", height=3) app.display()

7. master: Container của widget.

from guizero import App, Box, Text app = App("Example: master") box = Box(app) message = Text(box, text="Hello, World!") app.display()

8. text_size: Kích thước của văn bản.

from guizero import App, Text app = App("Example: text_size") message = Text(app, text="Hello, World!", text_size=20) app.display()

9. text_color: Màu của văn bản.

from guizero import App, Text app = App("Example: text_color") message = Text(app, text="Hello, World!", text_color="blue") app.display()

10. text_bold: Văn bản có in đậm hay không.

from guizero import App, Text app = App("Example: text_bold") message = Text(app, text="Hello, World!", text_bold=True) app.display()

11. text_italic: Văn bản có in nghiêng hay không.

from guizero import App, Text app = App("Example: text_italic") message = Text(app, text="Hello, World!", text_italic=True) app.display()

12. text_underline: Văn bản có gạch chân hay không.

from guizero import App, Text app = App("Example: text_underline") message = Text(app, text="Hello, World!", text_underline=True) app.display()

13. text_overstrike: Văn bản có gạch ngang hay không.

from guizero import App, Text app = App("Example: text_overstrike") message = Text(app, text="Hello, World!", text_overstrike=True) app.display()

14. tk: Đối tượng tkinter nội bộ.

from guizero import App, Text app = App("Example: tk") message = Text(app, text="Hello, World!") tk_obj = message.tk app.display()

15. value: Văn bản của widget.

from guizero import App, Text app = App("Example: value") message = Text(app, text="Hello, World!") print(message.value) app.display()

16. visible: Trạng thái hiển thị của widget.

from guizero import App, Text app = App("Example: visible") message = Text(app, text="Hello, World!", visible=False) app.display()

17. width: Chiều rộng của widget.

from guizero import App, Text app = App("Example: width") message = Text(app, text="Hello, World!", width=20) app.display()
