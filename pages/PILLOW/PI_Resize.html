Giới thiệu Pillow

Pillow là một thư viện mạnh mẽ để xử lý hình ảnh, cho phép:

  • Mở và chỉnh sửa hình ảnh với nhiều định dạng (JPEG, PNG, BMP...).
  • Thao tác hình ảnh: thay đổi kích thước, cắt, xoay, áp dụng bộ lọc, vẽ lên hình ảnh...
  • Chuyển đổi giữa các định dạng hình ảnh.
  • Hỗ trợ xử lý pixel theo từng cấp độ.

Cài đặt

pip install pillow

Các thành phần chính của Pillow

  1. Mở hình ảnh (Image.open): Dùng để mở tệp hình ảnh từ máy tính.
  2. Hiển thị hình ảnh (Image.show): Xem hình ảnh trực tiếp bằng trình xem ảnh mặc định.
  3. Thao tác hình ảnh:
    • Thay đổi kích thước: resize()
    • Cắt (crop): crop()
    • Xoay: rotate()
    • Lật: transpose()
  4. Vẽ lên hình ảnh (ImageDraw): Dùng để thêm văn bản, hình dạng (hình tròn, hình chữ nhật...).
  5. Áp dụng bộ lọc (ImageFilter): Làm mờ, làm sắc nét, phát hiện cạnh.
  6. Lưu hình ảnh (save): Lưu hình ảnh chỉnh sửa về máy.

Sử dụng Pillow kết hợp guizero

Guizero không hỗ trợ trực tiếp xử lý hình ảnh, nhưng bạn có thể kết hợp nó với Pillow để hiển thị hoặc thao tác hình ảnh trước khi hiển thị trong GUI.

1. Mở và hiển thị hình ảnh

Chức năng:

  • Mở một hình ảnh từ file.
  • Hiển thị hình ảnh trên cửa sổ mặc định.

Cách thực hiện:

Sử dụng các phương pháp cơ bản từ thư viện Pillow (Image).

1. Ví dụ

pip install pillow from PIL import Image # Mở file ảnh image = Image.open("example.jpg") # Hiển thị ảnh image.show() # Lấy thông tin về ảnh print(f"Size: {image.size}, Format: {image.format}, Mode: {image.mode}") 2. Thay đổi kích thước ảnh trước khi hiển thị

Chức năng:

  • Thay đổi kích thước hình ảnh bằng cách
    1. Giữ nguyên tỷ lệ.
    2. Bỏ tỷ lệ và thay đổi kích thước chính xác.
  • Sử dụng phương pháp resize() hoặc thumbnail().

Cách 1: Thay đổi kích thước giữ nguyên tỷ lệ (thumbnail)

from guizero import App, Picture from PIL import Image # Xử lý ảnh với Pillow image = Image.open("GUI/Source code/chuong5/topic_5/image1.png") # Resize ảnh max_size = (200, 1000) image.thumbnail(max_size) # Chuyển đổi sang chế độ RGB nếu cần thiết if image.mode == "RGBA": image = image.convert("RGB") # Lưu ảnh đã xử lý image.save("processed_image.jpg") # Lưu ảnh dưới dạng JPEG # Hiển thị trong guizero app = App("Hiển thị ảnh") picture = Picture(app, image="processed_image.jpg") app.display()

Khi bạn sử dụng phương thức thumbnail() trong Pillow và cung cấp một kích thước (chiều rộng và chiều cao) không tỷ lệ với hình ảnh gốc, thì hình ảnh sẽ tự động được thu nhỏ và giữ nguyên tỷ lệ gốc.

  • Pillow không chỉnh sửa hình ảnh để vừa chính xác với kích thước bạn đưa ra. Thay vào đó, nó thu nhỏ hình ảnh sao cho chiều rộng hoặc chiều cao không vượt quá kích thước tối đa bạn chỉ định.
  • Một chiều (chiều rộng hoặc chiều cao) sẽ nhỏ hơn hoặc bằng giá trị tối đa bạn cung cấp, nhưng tỷ lệ ban đầu của hình ảnh sẽ được giữ nguyên.

Ví dụ

Giả sử bạn có một hình ảnh gốc với kích thước 800x600 (tỷ lệ 4:3) và bạn đặt kích thước thumbnail là (200, 200).

from PIL import Image # Mở hình ảnh gốc img = Image.open("input.jpg") # Kích thước thumbnail thumbnail_size = (200, 200) # Tạo thumbnail (giữ nguyên tỷ lệ) img.thumbnail(thumbnail_size) # Hiển thị kích thước mới print("Kích thước gốc:", img.size) # Output: (200, 150)

Giải thích:

  • Tỷ lệ gốc của hình ảnh là 4:3
  • Khi thu nhỏ hình ảnh để vừa với (200, 200), Pillow tự động tính toán:
    1. Chiều rộng được thu nhỏ xuống 200px.
    2. Chiều cao được tính theo tỷ lệ và trở thành 150px.

Kết quả: Hình ảnh mới có kích thước 200x150, vì nếu thay đổi chiều cao thành 200px thì chiều rộng sẽ vượt quá giới hạn.


Trường hợp nếu cung cấp chiều cao và chiều rộng lệch

Nếu bạn cung cấp một giá trị lệch hẳn so với tỷ lệ ảnh gốc, Pillow vẫn giữ nguyên tỷ lệ gốc và chỉ giảm kích thước đến mức vừa nhất.

Ví dụ

Giả sử hình ảnh gốc là 1000x500 (tỷ lệ 2:1) và bạn đặt thumbnail_size = (300, 400).

img = Image.open("input.jpg") img.thumbnail((300, 400)) print("Kích thước sau khi thu nhỏ:", img.size)

Kết quả:

  • Pillow sẽ giảm chiều rộng xuống 300px (giới hạn tối đa).
  • Chiều cao sẽ tự động điều chỉnh theo tỷ lệ gốc (2:1), nên trở thành 150px.
  • Kết quả cuối cùng là hình ảnh có kích thước 300x150.

Quy tắc hoạt động của thumbnail():

  1. Giữ nguyên tỷ lệ gốc của hình ảnh.
  2. Giảm kích thước đến mức tối đa sao cho hình ảnh vừa với chiều rộng và chiều cao mà bạn cung cấp.
  3. Không bao giờ làm méo hình ảnh, vì vậy, nếu tỷ lệ không khớp với kích thước bạn đặt, kích thước cuối cùng sẽ nhỏ hơn giới hạn ở ít nhất một chiều.

Nếu bạn muốn hình ảnh bị bóp méo để vừa chính xác với kích thước bạn đưa ra, bạn cần sử dụng resize() thay vì thumbnail():


Cách 2: Thay đổi kích thước chính xác (resize)

Phương thức resize() trong Pillow được sử dụng để thay đổi kích thước hình ảnh một cách chính xác, không giữ nguyên tỷ lệ gốc trừ khi bạn xử lý riêng.

from PIL import Image from guizero import App, Picture # Mở hình ảnh img = Image.open("GUI/Source code/chuong5/topic_5/image1.png") # Kích thước mới new_size = (200, 300) # Thay đổi kích thước resized_img = img.resize(new_size, Image.Resampling.LANCZOS) # Kiểm tra mode và chuyển đổi nếu cần if resized_img.mode == "RGBA": resized_img = resized_img.convert("RGB") # Chuyển đổi sang RGB để lưu dưới dạng JPEG # Lưu hình ảnh sau khi resize resized_img.save("output_resized.jpg", format="JPEG") # Hiển thị kích thước mới app = App("Hiển thị ảnh") picture = Picture(app, image="output_resized.jpg") app.display()

Hoạt động của resize()

  1. resize() ép hình ảnh về kích thước bạn cung cấp, bất kể tỷ lệ gốc. Điều này có thể làm méo hoặc kéo dài hình ảnh nếu tỷ lệ mới không khớp với tỷ lệ ban đầu.
  2. Bạn có thể sử dụng tham số Image.Resampling (trước đây là Image.ANTIALIAS) để đảm bảo chất lượng hình ảnh sau khi thay đổi kích thước.

Cú pháp

resize(size, resample=Image.Resampling.LANCZOS)
  • size: Một tuple (width, height) với chiều rộng (width) và chiều cao (height) mới.
  • resample: Thuật toán nội suy được sử dụng để thay đổi kích thước. Một số tùy chọn:
    1. Image.Resampling.NEAREST: Nội suy gần nhất (chất lượng thấp, nhanh).
    2. Image.Resampling.BOX: Nội suy hộp (chất lượng trung bình).
    3. Image.Resampling.BILINEAR: Nội suy tuyến tính (chất lượng trung bình).
    4. Image.Resampling.HAMMING: Nội suy Hamming (chất lượng cao).
    5. Image.Resampling.BICUBIC: Nội suy bicubic (chất lượng cao hơn).
    6. Image.Resampling.LANCZOS: Nội suy Lanczos (chất lượng tốt nhất, thường dùng).
from PIL import Image from guizero import App, Picture # Mở hình ảnh gốc img = Image.open("GUI/Source code/chuong5/topic_5/image1.png") # Kích thước mới new_size = (200, 200) # Tạo các phiên bản của hình ảnh với các phương pháp nội suy khác nhau resized_nearest = img.resize(new_size, resample=Image.Resampling.NEAREST) resized_box = img.resize(new_size, resample=Image.Resampling.BOX) resized_bilinear = img.resize(new_size, resample=Image.Resampling.BILINEAR) resized_hamming = img.resize(new_size, resample=Image.Resampling.HAMMING) resized_bicubic = img.resize(new_size, resample=Image.Resampling.BICUBIC) resized_lanczos = img.resize(new_size, resample=Image.Resampling.LANCZOS) if resized_nearest.mode == "RGBA" and resized_box.mode == "RGBA" and resized_bilinear.mode == "RGBA" and resized_hamming.mode == "RGBA" and resized_bicubic.mode == "RGBA" and resized_lanczos.mode == "RGBA": resized_nearest = resized_nearest.convert("RGB") resized_box = resized_box.convert("RGB") resized_bilinear = resized_bilinear.convert("RGB") resized_hamming = resized_hamming.convert("RGB") resized_bicubic = resized_bicubic.convert("RGB") resized_lanczos = resized_lanczos.convert("RGB") # Lưu các hình ảnh để so sánh resized_nearest.save("output_nearest.jpg", format="JPEG") resized_box.save("output_box.jpg", format="JPEG") resized_bilinear.save("output_bilinear.jpg", format="JPEG") resized_hamming.save("output_hamming.jpg", format="JPEG") resized_bicubic.save("output_bicubic.jpg", format="JPEG") resized_lanczos.save("output_lanczos.jpg", format="JPEG") app = App("Hiển thị ảnh") picture = Picture(app, image="output_nearest.jpg") picture = Picture(app, image="output_box.jpg") picture = Picture(app, image="output_bilinear.jpg") picture = Picture(app, image="output_hamming.jpg") picture = Picture(app, image="output_bicubic.jpg") picture = Picture(app, image="output_lanczos.jpg") app.display()
  1. Image.Resampling.NEAREST:
    • Nội suy gần nhất, chọn giá trị màu từ pixel gần nhất mà không cần tính toán trung gian.
    • Chất lượng thấp, nhanh nhất. Có thể xuất hiện răng cưa, phù hợp với hình ảnh dạng pixel art hoặc khi cần tốc độ cao.
    • Ứng dụng: Hình ảnh không quan trọng về chất lượng, tốc độ ưu tiên.

      • Kết quả: Hình ảnh có thể xuất hiện khối màu cứng, không mượt.

  2. Image.Resampling.BOX:
    • Tính trung bình giá trị pixel trong một hộp (box) xung quanh mỗi pixel.
    • Chất lượng trung bình, nhanh hơn các thuật toán cao cấp.
    • Ứng dụng: Hình ảnh kích thước nhỏ hoặc vừa, không yêu cầu độ chi tiết cao.

      • Kết quả: Hình ảnh tương đối mượt nhưng không sắc nét.

  3. Image.Resampling.BILINEAR:
    • Tính trung bình hai chiều (x và y) giữa các pixel gần nhất.
    • Chất lượng trung bình, mượt hơn NEAREST, nhưng chi tiết có thể bị mất.
    • Ứng dụng: Hình ảnh nhỏ gọn cần sự mượt mà mà không đòi hỏi chi tiết cao.

      • Kết quả: Hình ảnh mượt mà hơn BOX, nhưng thiếu sắc nét.

  4. Image.Resampling.HAMMING:
    • Một phương pháp mượt hóa cao hơn, giảm độ nhiễu trong khi giữ được chi tiết.
    • Chất lượng tốt, phù hợp với ảnh nhỏ hoặc trung bình.
    • Ứng dụng: Khi cần chất lượng tốt cho ảnh nhỏ hoặc trung bình.

      • Kết quả: Hình ảnh chất lượng cao, mượt và giảm nhiễu.

  5. Image.Resampling.BICUBIC:
    • Tính toán dựa trên 16 pixel xung quanh mỗi pixel mới, cho phép chi tiết tốt hơn.
    • Chất lượng cao hơn BILINEAR.
    • Ứng dụng: Thay đổi kích thước hình ảnh chất lượng cao.

      • Kết quả: Hình ảnh sắc nét, mượt, phù hợp với nội dung chi tiết.

  6. Image.Resampling.LANCZOS:
    • Sử dụng một hàm sinc để lấy mẫu lại, mang lại chất lượng tốt nhất.
    • Tốn tài nguyên tính toán nhất.
    • Ứng dụng: Khi cần chất lượng tốt nhất cho hình ảnh lớn hoặc cần giữ lại nhiều chi tiết.

      • Kết quả: Hình ảnh mượt và chi tiết nhất, lý tưởng cho ảnh lớn.

Cắt ảnh (crop)

  1. Cắt một phần hình ảnh bằng cách cung cấp tọa độ (x, y) của vùng cần cắt.
  2. Dữ liệu đầu vào là một hình chữ nhật với 4 tọa độ: (left, top, right, bottom).
  3. Trả về một ảnh mới chứa phần đã cắt.

Cú pháp

cropped_image = image.crop((x1, y1, x2, y2)) from PIL import Image, ImageDraw from guizero import App, Picture, Text # Mở ảnh có sẵn img = Image.open("GUI/Source code/chuong5/topic_5/image1.png") # Lấy kích thước ban đầu của ảnh original_size = img.size # (width, height) # Xác định vùng cắt crop_area = (50, 50, 300, 300) # (left, top, right, bottom) # Cắt ảnh cropped_image = img.crop(crop_area) # Lấy kích thước sau khi cắt cropped_size = cropped_image.size # (width, height) # Tạo ứng dụng guizero app = App("Vẽ văn bản lên ảnh tải lên") # Hiển thị kích thước trước khi cắt Text(app, text=f"Kích thước ban đầu: {original_size[0]} x {original_size[1]}", size=14) # Hiển thị kích thước sau khi cắt Text(app, text=f"Kích thước sau khi cắt: {cropped_size[0]} x {cropped_size[1]}", size=14) # Hiển thị ảnh đã cắt picture = Picture(app, image=cropped_image) app.display()

Xoay ảnh (rotate)

  1. Xoay ảnh theo góc độ bất kỳ (đơn vị: độ).
  2. Mặc định, Pillow xoay ảnh theo ngược chiều kim đồng hồ.

Cú pháp

rotated_image = image.rotate(angle, expand=True)
  • angle: Góc xoay (độ). Dương = ngược chiều kim đồng hồ, âm = xuôi chiều kim đồng hồ.
  • expand=True: Tự động mở rộng ảnh nếu bị cắt khi xoay.

Ví dụ 1: Xoay ảnh 45 độ

from PIL import Image # Mở ảnh image = Image.open("example.jpg") # Xoay ảnh 45 độ rotated_image = image.rotate(45, expand=True) # Hiển thị ảnh sau khi xoay rotated_image.show() # Lưu ảnh đã xoay rotated_image.save("rotated_45.jpg")

Ví dụ 2: Xoay ảnh 90 độ và giữ nguyên kích thước

rotated_image = image.rotate(90, expand=False) rotated_image.show()

Lật ảnh (transpose)

  1. Lật ảnh theo nhiều hướng khác nhau: lật ngang, lật dọc, xoay 90 độ, 180 độ, 270 độ.
  2. Sử dụng phương thức transpose().

Các kiểu lật ảnh

Phương thức Mô tả
Image.FLIP_LEFT_RIGHT Lật ngang (phản chiếu gương)
Image.FLIP_TOP_BOTTOM Lật dọc
Image.ROTATE_90 Xoay 90 độ
Image.ROTATE_180 Xoay 180 độ
Image.ROTATE_270 Xoay 270 độ

Ví dụ 1: Lật ảnh ngang (gương phản chiếu)

from PIL import Image # Mở ảnh image = Image.open("example.jpg") # Lật ảnh ngang flipped_image = image.transpose(Image.FLIP_LEFT_RIGHT) # Hiển thị ảnh đã lật flipped_image.show() # Lưu ảnh flipped_image.save("flipped_horizontal.jpg")

Ví dụ 2: Lật ảnh dọc

flipped_image = image.transpose(Image.FLIP_TOP_BOTTOM) flipped_image.show()

Ví dụ 3: Xoay ảnh 90, 180, 270 độ

rotated_90 = image.transpose(Image.ROTATE_90) rotated_180 = image.transpose(Image.ROTATE_180) rotated_270 = image.transpose(Image.ROTATE_270) rotated_90.show() rotated_180.show() rotated_270.show()
3. Áp dụng bộ lọc lên ảnh

Trong thư viện Pillow, bộ lọc ảnh (ImageFilter) cung cấp nhiều tùy chọn để xử lý và cải thiện hình ảnh. Dưới đây là toàn bộ các bộ lọc ảnh phổ biến trong Pillow, kèm ví dụ chi tiết.

Danh sách các bộ lọc trong Pillow

1. Làm mờ

  • Chức năng: Làm mờ toàn bộ hình ảnh.
  • Sử dụng: ImageFilter.BLUR
    • from PIL import Image, ImageFilter # Mở ảnh image = Image.open("example.jpg") # Áp dụng bộ lọc làm mờ blurred_image = image.filter(ImageFilter.BLUR) # Lưu ảnh làm mờ blurred_image.save("blurred_image.jpg") blurred_image.show()

2. Làm sắc nét (SHARPEN)

  • Chức năng: Tăng độ sắc nét của hình ảnh.
  • Sử dụng: ImageFilter.SHARPEN
    • from PIL import Image, ImageFilter # Mở ảnh image = Image.open("example.jpg") # Áp dụng bộ lọc làm sắc nét sharpened_image = image.filter(ImageFilter.SHARPEN) # Lưu ảnh sau khi làm sắc nét sharpened_image.save("sharpened_image.jpg") sharpened_image.show()

3. Làm mờ Gaussian (GAUSSIANBLUR)

  • Chức năng: Làm mờ hình ảnh bằng phương pháp Gaussian.
  • Sử dụng: ImageFilter.GaussianBlur(radius - độ mạnh của hiệu ứng mờ (mặc định là 2.0).
    • from PIL import Image, ImageFilter # Mở ảnh image = Image.open("example.jpg") # Áp dụng bộ lọc làm mờ Gaussian với bán kính 5 gaussian_blur = image.filter(ImageFilter.GaussianBlur(5)) # Lưu ảnh làm mờ Gaussian gaussian_blur.save("gaussian_blur_image.jpg") gaussian_blur.show()

4. Làm mịn (SMOOTH)

  • Sử dụng: ImageFilter.DETAIL

  • Sử dụng: ImageFilter.CONTOUR

9. Tạo ảnh nổi (EMBOSS)

  • Chức năng: Chuyển đổi ảnh thành dạng nổi 3D (giống khắc trên đá).
  • Sử dụng: ImageFilter.EMBOSS
    • from PIL import Image, ImageFilter # Mở ảnh image = Image.open("example.jpg") # Áp dụng bộ lọc Emboss embossed_image = image.filter(ImageFilter.EMBOSS) # Lưu ảnh embossed_image.save("embossed_image.jpg") embossed_image.show()

10. Tùy chỉnh bộ lọc với Kernel

  • Chức năng: Tạo bộ lọc tùy chỉnh bằng ma trận (Kernel). Thường dùng để làm mờ, làm sắc nét hoặc phát hiện cạnh.
    • from PIL import Image, ImageFilter # Mở ảnh image = Image.open("example.jpg") # Tạo kernel tùy chỉnh (làm sắc nét) kernel = ImageFilter.Kernel((3, 3), [-1, -1, -1, -1, 9, -1, -1, -1, -1], scale=None) # Áp dụng kernel tùy chỉnh custom_filtered_image = image.filter(kernel) # Lưu ảnh custom_filtered_image.save("custom_filtered_image.jpg") custom_filtered_image.show()

    Các tham số của ImageFilter.Kernel:

    • (3, 3): Kích thước kernel (3×3).
    • [-1, -1, -1, -1, 9, -1, -1, -1, -1]: Ma trận giá trị ảnh hưởng đến pixel.
    • scale=None: Khi để None, Pillow sẽ tự động chuẩn hóa giá trị pixel để không bị quá sáng hoặc tối.

    Giải thích giá trị kernel:

    -1 -1 -1 -1 9 -1 -1 -1 -1
    • Tổng các giá trị trong kernel là 1, nghĩa là ảnh không bị thay đổi độ sáng trung bình.
    • Giá trị 9 tại trung tâm làm nổi bật pixel trung tâm.
    • Các giá trị -1 xung quanh giúp tăng độ tương phản, tạo hiệu ứng làm sắc nét (sharpen).
