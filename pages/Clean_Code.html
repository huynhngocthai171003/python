CLEAN CODE TRONG PYTHON

Clean Code là cách viết mã nguồn sao cho:

  • Dễ đọc: Người khác (hoặc chính bạn trong tương lai) có thể hiểu ý nghĩa và mục đích của code một cách dễ dàng.
  • Dễ bảo trì: Khi cần sửa đổi hoặc mở rộng, code không gây rối và giảm nguy cơ phát sinh lỗi.
  • Đơn giản: Giải quyết vấn đề theo cách trực quan, tránh phức tạp không cần thiết.

Good Documentation (Comment)

Vai trò của comment:

  • Giải thích các đoạn code phức tạp.
  • Chỉ ra ý định và cách hoạt động của code.

Nguyên tắc:

  • Không viết những comment dư thừa (code dễ hiểu không cần comment).
  • Comment tập trung vào "tại sao" hơn là "như thế nào".

1. Ví dụ: # Bad Comment x = 10 # Assigning value 10 to x # Good Comment # Calculate the total price after discount total_price = calculate_discount(price, discount_rate) 2. Thực hành: Thêm comment giải thích cho đoạn code sau: def calculate_total(items): total = 0 for item in items: total += item['price'] return total

Meaningful Variable and Function Names

Nguyên tắc đặt tên:

  • Tên biến phải mô tả chính xác chức năng của nó.
  • Tên hàm nên biểu thị rõ nhiệm vụ của hàm.

Cần tránh:

  • Tên chung chung như x, temp, data.
  • Tên quá dài hoặc viết tắt không rõ nghĩa.

1. Ví dụ: # Bad Naming def calc(x, y): return x + y # Good Naming def calculate_sum(first_number, second_number): return first_number + second_number 2. Thực hành: Tối ưu tên biến và hàm cho đoạn code sau: def f(a, b): c = a * b return c result = f(10, 20) print(result)

Modular Code (Split Code)

Định nghĩa Modular Code: Chia nhỏ chương trình thành các hàm hoặc module độc lập.

Lợi ích:

  • Dễ kiểm tra và bảo trì.
  • Tái sử dụng code.

Quy tắc:

  • Mỗi hàm/module chỉ nên đảm nhận một nhiệm vụ duy nhất (Single Responsibility Principle).

1. Ví dụ: # Without Modular Code def process_data(data): # Validate data if not data: return None # Process data processed_data = [item**2 for item in data] return processed_data # With Modular Code def validate_data(data): return bool(data) def process_data(data): return [item**2 for item in data] # Main code data = [1, 2, 3] if validate_data(data): result = process_data(data) print(result) 2. Thực hành: Tách đoạn code sau thành các hàm nhỏ hơn: def main(): numbers = [1, 2, 3, 4, 5] even_numbers = [] for num in numbers: if num % 2 == 0: even_numbers.append(num) print(even_numbers) main()

Bài tập

Bài 1: Tính Tiền Giảm Giá (Good Documentation)

Đề bài: Viết comment giải thích ý định và cách hoạt động cho đoạn code sau:

def calculate_discounted_price(price, discount_rate): discounted_price = price - (price * discount_rate / 100) return discounted_price original_price = 1000 discount = 20 final_price = calculate_discounted_price(original_price, discount) print(final_price)

Bài 2: Tính Điểm Trung Bình (Modular)

Đề bài: Viết lại đoạn code sau sao cho sử dụng hàm và tổ chức code theo module:

grades = [50, 70, 90] total = sum(grades) average = total / len(grades) if average >= 60: print("Pass") else: print("Fail")

Bài 3: Quản Lý Sinh Viên

Đề bài: Viết chương trình quản lý danh sách sinh viên trong một lớp học. Chương trình cần có các chức năng:

  1. Thêm sinh viên mới (họ tên, mã số, điểm).
  2. Hiển thị danh sách sinh viên.
  3. Tìm sinh viên theo mã số.
  4. Tính điểm trung bình của lớp.
  5. Hiển thị danh sách sinh viên xếp loại (giỏi, khá, trung bình, yếu).
