Biến trong Python

Biến là một phần quan trọng trong lập trình Python, cho phép chúng ta lưu trữ, thao tác và tham chiếu dữ liệu trong suốt quá trình thực thi chương trình. Bài viết này sẽ hướng dẫn chi tiết về biến trong Python, bao gồm các chủ đề như biến trong Python, địa chỉ bộ nhớ, tạo biến, in biến, xóa biến, lấy kiểu của biến, ép kiểu biến, phân biệt chữ hoa chữ thường, gán nhiều biến, quy tắc đặt tên biến, biến cục bộ, biến toàn cục, và hằng số trong Python.


Biến trong Python

Biến trong Python là các vị trí bộ nhớ được dành riêng để lưu trữ giá trị. Khi bạn tạo một biến, bạn đang dành một không gian trong bộ nhớ để lưu trữ giá trị đó. Python không yêu cầu khai báo biến trước khi sử dụng hoặc khai báo kiểu của biến. Một biến được tạo ra ngay khi bạn gán giá trị cho nó.

x = 5 y = "John" print(x) print(y)

Mỗi biến trong Python là một tên đại diện cho một vị trí bộ nhớ.

x = 5 print(id(x))

Python không có lệnh để khai báo biến.

x = 4 # x là kiểu int x = "Sally" # x bây giờ là kiểu str print(x)

Bạn có thể sử dụng hàm print() để in giá trị của biến.

x = 5 y = "John" print(x) print(y)

Bạn có thể xóa một biến bằng cách sử dụng lệnh del.

x = 5 print(x) del x # print(x) # Sẽ gây ra lỗi vì biến x đã bị xóa

Sử dụng hàm type() để lấy kiểu của biến.

x = 5 y = "John" print(type(x)) print(type(y))

Ép kiểu biến:

x = str(3) y = int(3) z = float(3) print(x) print(y) print(z)

Tên biến trong Python phân biệt chữ hoa chữ thường.

a = 4 A = "Sally" print(a) print(A)

Gán nhiều biến trong một dòng.

x, y, z = "Orange", "Banana", "Cherry" print(x) print(y) print(z) x = y = z = "Orange" print(x) print(y) print(z)

Tên biến hợp lệ:

user_age = 20 website = 'https://freecodecamp.org' # 1password = True # không hợp lệ

Biến cục bộ:

def sum(x, y): sum = x + y return sum print(sum(5, 10))

Biến toàn cục:

x = 5 y = 10 def sum(): return x + y print(sum())

Hằng số trong Python (quy ước dùng chữ in hoa):

PI = 3.14159 GRAVITY = 9.8

Việc nắm vững cách sử dụng biến trong Python là rất quan trọng để trở thành một lập trình viên Python thành thạo. Bài viết này đã giới thiệu chi tiết về các khái niệm cơ bản như biến trong Python, địa chỉ bộ nhớ, tạo biến, in biến, xóa biến, lấy kiểu của biến, ép kiểu biến, phân biệt chữ hoa chữ thường, gán nhiều biến, quy tắc đặt tên biến, biến cục bộ, biến toàn cục, và hằng số trong Python. Hy vọng rằng bạn đã có thể hiểu rõ hơn về cách sử dụng biến trong Python và sẵn sàng viết mã Python của riêng mình.

Kiểu dữ liệu trong Python

5 phút đọc

Kiểu dữ liệu là một khái niệm quan trọng trong lập trình, giúp xác định loại dữ liệu mà một biến có thể lưu trữ và các phép toán có thể thực hiện trên dữ liệu đó. Python là một ngôn ngữ lập trình mạnh mẽ và linh hoạt, hỗ trợ nhiều kiểu dữ liệu khác nhau.

Kiểu dữ liệu là gì?

Kiểu dữ liệu (data type) là một phân loại của các loại dữ liệu cụ thể bởi một giá trị nhất định hoặc các phép toán toán học hoặc logic có thể thực hiện trên dữ liệu đó. Trong Python, mọi thứ đều là đối tượng.

Các kiểu dữ liệu trong Python

Dưới đây là một số kiểu dữ liệu phổ biến trong Python:

  1. Numeric (Số học): int, float, complex
  2. String (Chuỗi): str
  3. Sequence (Dãy): list, tuple, range
  4. Binary (Nhị phân): bytes, bytearray, memoryview
  5. Mapping (Ánh xạ): dict
  6. Boolean (Boolean): bool
  7. Set (Tập hợp): set, frozenset
  8. None (Không có giá trị): NoneType

Kiểu dữ liệu số (Numeric Data Types)

Kiểu dữ liệu số trong Python được sử dụng để lưu trữ các giá trị liên quan đến số học. Các giá trị này bao gồm số nguyên, số thực và số phức. Python hỗ trợ ba kiểu dữ liệu số chính:

1. int (Số nguyên)

Lưu trữ các số nguyên, tức là những số không có phần thập phân. Kiểu int có thể biểu diễn các giá trị rất lớn vì không giới hạn độ dài.

Ví dụ:

num1 = 5 print(num1, 'là kiểu', type(num1))

2. float (Số thực)

Dùng để lưu trữ các số có phần thập phân. Python hỗ trợ độ chính xác lên đến khoảng 15 chữ số thập phân cho kiểu float.

Ví dụ:

num2 = 2.0 print(num2, 'là kiểu', type(num2))

3. complex (Số phức)

Được sử dụng để biểu diễn số phức, có dạng a + bj trong đó a và b là các số thực, còn j là đơn vị ảo (tương đương với √-1).

Ví dụ:

num3 = 1 + 2j print(num3, 'là kiểu', type(num3))

Kiểu dữ liệu chuỗi (String Data Type)

Kiểu dữ liệu chuỗi trong Python được sử dụng để lưu trữ văn bản, hay còn gọi là chuỗi ký tự. Một chuỗi là một tập hợp có thứ tự các ký tự (letters, numbers, symbols), được đặt trong dấu ngoặc đơn ' ' hoặc dấu ngoặc kép " ".

Python hỗ trợ các ký tự Unicode, cho phép bạn làm việc với nhiều ngôn ngữ và biểu tượng trên thế giới. Chuỗi là một kiểu dữ liệu bất biến (immutable), nghĩa là giá trị chuỗi không thể thay đổi sau khi được tạo.

Ví dụ:

name = 'Python' print(name) message = "Python for beginners" print(message)

Kiểu dữ liệu dãy (Sequence Data Types)

Kiểu dữ liệu dãy trong Python được sử dụng để lưu trữ một tập hợp các mục (elements) theo thứ tự. Các phần tử trong dãy có thể được truy cập thông qua chỉ số (index), bắt đầu từ 0.

Python hỗ trợ ba kiểu dữ liệu dãy phổ biến:

1. list (Danh sách)

list là một dãy có thể thay đổi (mutable), cho phép thêm, xóa hoặc thay đổi phần tử sau khi được tạo. Các phần tử được đặt trong dấu ngoặc vuông [] và có thể thuộc nhiều kiểu dữ liệu khác nhau.

Ví dụ:

fruits = ["apple", "banana", "cherry"] print(fruits)

2. tuple (Bộ giá trị không thay đổi)

tuple là một dãy bất biến (immutable), tức là không thể thay đổi nội dung sau khi được tạo. Các phần tử được đặt trong dấu ngoặc đơn ().

Ví dụ:

coordinates = (10.0, 20.0) print(coordinates)

3. range (Dãy số nguyên)

range đại diện cho một dãy các số nguyên liên tiếp, thường được sử dụng trong vòng lặp for. Mặc định, range(n) sẽ tạo ra các số từ 0 đến n-1.

Ví dụ:

numbers = range(5) print(list(numbers))

Kiểu dữ liệu nhị phân (Binary Data Types)

Python hỗ trợ các kiểu dữ liệu nhị phân dùng để làm việc với dữ liệu dạng byte – thường được sử dụng trong xử lý tệp, truyền dữ liệu qua mạng, hoặc thao tác cấp thấp với bộ nhớ.

Ba kiểu dữ liệu nhị phân chính trong Python là:

1. bytes

bytes là một dãy các byte không thể thay đổi (immutable). Mỗi phần tử trong một đối tượng bytes là một số nguyên trong khoảng từ 0 đến 255.

Ví dụ:

data = b"Hello" print(data)

2. bytearray

bytearray tương tự như bytes nhưng có thể thay đổi được (mutable). Điều này cho phép sửa đổi dữ liệu sau khi đã tạo đối tượng.

Ví dụ:

data = bytearray(5) print(data)

3. memoryview

memoryview cung cấp một giao diện cho phép thao tác trực tiếp trên bộ nhớ của các đối tượng hỗ trợ buffer protocol mà không cần sao chép dữ liệu, giúp tối ưu hiệu suất.

Ví dụ:

data = memoryview(bytes(5)) print(data)

Kiểu dữ liệu ánh xạ (Mapping Data Type)

Kiểu dữ liệu ánh xạ trong Python đại diện cho tập hợp các cặp khóa - giá trị (key-value). Kiểu phổ biến nhất là dictionary, được sử dụng để lưu trữ và truy xuất dữ liệu dựa trên khóa thay vì chỉ số.

Trong dict, mỗi khóa là duy nhất và được ánh xạ tới một giá trị cụ thể. Các cặp khóa-giá trị được đặt trong dấu ngoặc nhọn {}, phân cách bằng dấu hai chấm :, và các cặp được phân tách bằng dấu phẩy ,.

Ví dụ:

person = {"name": "John", "age": 30} print(person)

Kiểu dữ liệu boolean (Boolean Data Type)

Kiểu dữ liệu boolean trong Python dùng để biểu diễn giá trị đúng hoặc sai. Nó chỉ có hai giá trị hợp lệ là True và False – viết hoa chữ cái đầu và không có dấu nháy.

Kiểu boolean thường được sử dụng trong các biểu thức điều kiện (if, while, v.v.) và là kết quả trả về của các phép so sánh (==, >, <, ...).

Trong ngữ cảnh Boolean:

  • Các giá trị như 0, None, "", [], {} được coi là False.
  • Mọi giá trị khác đều được coi là True.

Ví dụ:

is_active = True print(is_active, 'là kiểu', type(is_active))

Kiểu dữ liệu tập hợp (Set Data Type)

Kiểu dữ liệu set trong Python được sử dụng để lưu trữ nhiều phần tử **duy nhất** trong một tập hợp **không có thứ tự** và **không cho phép trùng lặp**.

Một tập hợp được khai báo bằng dấu ngoặc nhọn {}, hoặc bằng hàm set(). Do không có thứ tự, các phần tử không thể được truy cập bằng chỉ số (index) như trong list hoặc tuple.

Tập hợp rất hữu ích khi cần loại bỏ các giá trị trùng lặp hoặc thực hiện các phép toán tập hợp như: hợp (union), giao (intersection), hiệu (difference).

Ví dụ:

student_ids = {112, 114, 116, 118, 115} print(student_ids)

Kiểu dữ liệu None (None Data Type)

None là một giá trị đặc biệt trong Python thuộc kiểu NoneType. Nó được sử dụng để biểu thị rằng một biến không có giá trị hoặc một hàm không trả về gì.

None thường được dùng trong các tình huống như:

  • Gán giá trị mặc định ban đầu cho biến.
  • Làm điều kiện kiểm tra xem một biến đã được khởi tạo hay chưa.
  • Làm giá trị trả về mặc định cho các hàm không có lệnh return.

Lưu ý rằng None không tương đương với False, 0 hoặc chuỗi rỗng "", nhưng khi kiểm tra logic, nó được coi là giá trị sai (falsy).

Ví dụ:

value = None print(value, 'là kiểu', type(value))

Chuyển đổi kiểu dữ liệu trong Python

Chuyển đổi kiểu dữ liệu là quá trình chuyển đổi giá trị từ kiểu dữ liệu này sang kiểu dữ liệu khác. Python hỗ trợ hai hình thức chuyển đổi:

  • Chuyển đổi ngầm định (Implicit Conversion)
  • Chuyển đổi tường minh (Explicit Conversion)

1. Chuyển đổi ngầm định (Implicit Type Conversion)

Chuyển đổi ngầm định xảy ra khi Python tự động thay đổi kiểu dữ liệu của một giá trị mà không cần sự can thiệp của lập trình viên. Điều này thường xảy ra trong các phép toán khi một kiểu có độ chính xác thấp được nâng lên kiểu có độ chính xác cao hơn để tránh mất dữ liệu.

Ví dụ:

a_int = 1 b_float = 1.0 c_sum = a_int + b_float print(c_sum) # Kết quả: 2.0 print(type(c_sum)) # Kết quả: <class 'float'>

2. Chuyển đổi tường minh (Explicit Type Conversion)

Chuyển đổi tường minh, hay còn gọi là ép kiểu (type casting), là khi lập trình viên chủ động thay đổi kiểu dữ liệu bằng cách sử dụng các hàm dựng sẵn như int(), float(), str(), v.v.

Ví dụ:

num_string = '12' num_integer = 23 print("Kiểu dữ liệu của num_string trước khi ép kiểu:", type(num_string)) # Ép kiểu num_string = int(num_string) print("Kiểu dữ liệu của num_string sau khi ép kiểu:", type(num_string)) num_sum = num_integer + num_string print("Tổng:", num_sum) print("Kiểu dữ liệu của num_sum:", type(num_sum))

Các hàm chuyển đổi kiểu dữ liệu phổ biến

  • int(): chuyển sang số nguyên
  • float(): chuyển sang số thực
  • str(): chuyển sang chuỗi
  • bool(): chuyển sang boolean
  • list(): chuyển sang danh sách
  • tuple(): chuyển sang tuple
  • set(): chuyển sang tập hợp
  • dict(): chuyển sang từ điển

Ví dụ sử dụng các hàm chuyển đổi

x = int("10") print(x, 'là kiểu', type(x)) y = float("10.5") print(y, 'là kiểu', type(y)) z = str(10) print(z, 'là kiểu', type(z)) a = bool(1) print(a, 'là kiểu', type(a)) b = list((1, 2, 3)) print(b, 'là kiểu', type(b)) c = tuple([1, 2, 3]) print(c, 'là kiểu', type(c)) d = set([1, 2, 3]) print(d, 'là kiểu', type(d)) e = dict([(1, 'one'), (2, 'two')]) print(e, 'là kiểu', type(e))
