Python Programming Language

Tổng hợp các hàm của List trong Python

Python là một ngôn ngữ lập trình mạnh mẽ và linh hoạt, được sử dụng rộng rãi trong nhiều lĩnh vực như phát triển web, khoa học dữ liệu, trí tuệ nhân tạo, và nhiều lĩnh vực khác. Một trong những cấu trúc dữ liệu quan trọng nhất trong Python là danh sách (list). Danh sách trong Python là một cấu trúc dữ liệu có thể chứa nhiều loại dữ liệu khác nhau và có thể thay đổi được. Trong bài viết này, chúng ta sẽ tìm hiểu chi tiết về các hàm và phương thức của danh sách trong Python, kèm theo các ví dụ minh họa.


Index()

Trả về chỉ số (vị trí) của phần tử đầu tiên khớp với giá trị chỉ định.

1. Cú pháp
list.index(x[, start[, end]]) Tham số: x: Phần tử cần tìm. start: (Tùy chọn) Vị trí bắt đầu tìm kiếm. end: (Tùy chọn) Vị trí kết thúc tìm kiếm. 2. Ví dụ:
my_list = [1, 2, 3, 2] print(my_list.index(2)) # Kết quả: 1 print(my_list.index(2, 2)) # Tìm từ vị trí thứ 2, Kết quả: 3

Copy()

Trả về một bản sao nông (shallow copy) của danh sách.

1. Cú pháp
new_list = list.copy() 2. Ví dụ:
original = [1, 2, 3] copied = original.copy() print(copied) # Kết quả: [1, 2, 3]

Count()

Trả về số lần xuất hiện của một phần tử trong danh sách.

1. Cú pháp
list.count(x) x: Phần tử cần đếm trong danh sách. 2. Ví dụ:
my_list = [1, 2, 2, 3, 4] print(my_list.count(2)) # Kết quả: 2

Reverse()

Đảo ngược thứ tự của các phần tử trong danh sách.

1. Cú pháp
list.reverse() 2. Ví dụ:
my_list = [1, 2, 3] my_list.reverse() print(my_list) # Kết quả: [3, 2, 1]

Sort()

Phương thức sort() là một phương thức của đối tượng list, được sử dụng để sắp xếp các phần tử của list tại chỗ (in-place). Điều này có nghĩa là list ban đầu sẽ bị thay đổi và không tạo ra một list mới.

1. Cú pháp
list.sort(key=None, reverse=False) key (tuỳ chọn): Một hàm có thể được sử dụng để tùy chỉnh thứ tự sắp xếp. reverse (tuỳ chọn): Một giá trị boolean. Nếu True, list sẽ được sắp xếp theo thứ tự giảm dần. Mặc định là False. 2. Ví dụ:
fruits = ['apple', 'banana', 'cherry'] fruits.sort() print(fruits) # Kết quả: ['apple', 'banana', 'cherry'] 3. Sắp xếp theo thứ tự giảm dần:
fruits = ['apple', 'banana', 'cherry'] fruits.sort(reverse=True) print(fruits) # Kết quả: ['cherry', 'banana', 'apple'] 4. Sắp xếp với hàm tùy chỉnh:
def myfunc(n): return abs(n - 50) numbers = [100, 50, 65, 82, 23] numbers.sort(key=myfunc) print(numbers) # Kết quả: [50, 65, 23, 82, 100]

Sorted()

Phương thức sorted() là một hàm tích hợp sẵn trong Python, được sử dụng để sắp xếp các phần tử của một iterable và trả về một list mới đã được sắp xếp. Hàm này không thay đổi iterable ban đầu

1. Cú pháp
sorted(iterable, key=None, reverse=False) iterable: Một đối tượng có thể lặp lại (list, tuple, dictionary, v.v.). key (tuỳ chọn): Một hàm có thể được sử dụng để tùy chỉnh thứ tự sắp xếp. reverse (tuỳ chọn): Một giá trị boolean. Nếu True, list sẽ được sắp xếp theo thứ tự giảm dần. Mặc định là False. 2. Ví dụ:
fruits = ['apple', 'banana', 'cherry'] sorted_fruits = sorted(fruits) print(sorted_fruits) # Kết quả: ['apple', 'banana', 'cherry'] print(fruits) # Kết quả: ['apple', 'banana', 'cherry']] 3. Sắp xếp theo thứ tự giảm dần:
fruits = ['apple', 'banana', 'cherry'] sorted_fruits = sorted(fruits, reverse=True) print(sorted_fruits) # Kết quả: ['cherry', 'banana', 'apple'] 4. Sắp xếp với hàm tùy chỉnh:
def myfunc(n): return abs(n - 50) numbers = [100, 50, 65, 82, 23] sorted_numbers = sorted(numbers, key=myfunc) print(sorted_numbers) # Kết quả: [50, 65, 23, 82, 100]

Sắp xếp List chứa các kiểu dữ liệu khác nhau

Python cho phép sắp xếp các list chứa các kiểu dữ liệu khác nhau như số nguyên, chuỗi, và thậm chí là các list khác. Tuy nhiên, tất cả các phần tử trong list phải có thể so sánh được với nhau.

1. Ví dụ
mixed_list = [3, 'banana', 1, 'apple', 2, 'cherry'] # mixed_list.sort() # Sẽ gây ra lỗi TypeError Để sắp xếp một list chứa các kiểu dữ liệu khác nhau, bạn có thể sử dụng tham số key để xác định tiêu chí sắp xếp.
2. Ví dụ:
mixed_list = [3, 'banana', 1, 'apple', 2, 'cherry'] sorted_mixed_list = sorted(mixed_list, key=str) print(sorted_mixed_list) # Kết quả: [1, 2, 3, 'apple', 'banana', 'cherry']

Min(), Max()

Trả về phần tử nhỏ nhất và lớn nhấttrong danh sách.

1. Cú pháp
min_value = min(list) max_value = max(list) 2. Ví dụ:
my_list = [3, 1, 4, 2] print(min(my_list)) # Kết quả: 1 print(max(my_list)) # Kết quả: 4

Enumerate()

Trả về một đối tượng liệt kê (iterator) chứa các cặp (chỉ số, giá trị) từ danh sách.

1. Cú pháp
enumerated_list = enumerate(list, start=0) list: Danh sách cần liệt kê. start: (Tùy chọn) Giá trị chỉ số bắt đầu. Mặc định là 0 2. Ví dụ:
my_list = ['a', 'b', 'c'] for index, value in enumerate(my_list): print(index, value) # Kết quả: # 0 a # 1 b # 2 c

All()

Trả về True nếu tất cả phần tử trong danh sách là True. Ngược lại trả về False.

1. Cú pháp
result = all(list) list: Danh sách chứa các phần tử cần kiểm tra. 2. Ví dụ:
my_list = [1, 2, 3] print(all(my_list)) # Kết quả: True my_list = [0, 2, 3] print(all(my_list)) # Kết quả: False

Any()

Trả về True nếu có ít nhất một phần tử trong danh sách là True. Nếu không, trả về False.

1. Cú pháp
result = any(list) list: Danh sách chứa các phần tử cần kiểm tra. 2. Ví dụ:
my_list = [0, 1, 2] print(any(my_list)) # Kết quả: True my_list = [0, 0, 0] print(any(my_list)) # Kết quả: False

Filter()

Lọc các phần tử của danh sách dựa trên một điều kiện cụ thể.

1. Cú pháp
filtered_list = filter(function, list) function: Hàm trả về True hoặc False cho từng phần tử. list: Danh sách cần lọc. 2. Ví dụ:
my_list = [1, 2, 3, 4, 5] def is_even(n): return n % 2 == 0 filtered = filter(is_even, my_list) print(list(filtered)) # Kết quả: [2, 4]

Map()

Áp dụng một hàm cho từng phần tử của danh sách và trả về danh sách kết quả.

1. Cú pháp
mapped_list = map(function, list) function: Hàm được áp dụng cho từng phần tử. list: Danh sách có các phần tử được áp dụng hàm 2. Ví dụ:
my_list = [1, 2, 3] def square(n): return n * 2 mapped = map(square, my_list) print(list(mapped)) # Kết quả: [2, 4, 6]
